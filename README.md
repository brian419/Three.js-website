## Website Overview
The website features interactive 3D scenes created with Three.js. Key sections include:

- **Home**: A welcoming space with a galaxy-themed background, where visitors can start their journey.
- **About Me**: Learn about my background, education, and what drives my passion for technology.
- **Projects**: A showcase of my work, demonstrating my skills in various programming languages and tools.
- **Contact**: Information on how to connect with me, including email and phone number.

## Technical Details
- **Three.js**: Utilized for rendering 3D graphics in the web browser.
- **Interactive Elements**: Implemented using Raycaster for mouse interactions, enhancing user engagement.
- **Responsive Design**: Ensures a seamless experience across different devices and screen sizes.

## How to Navigate
- Use the mouse to interact with 3D elements and navigate through the website.
- Click on the various sections like 'About Me', 'Projects', and 'Contact' to explore more.
